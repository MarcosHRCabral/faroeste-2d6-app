# Faroeste +2D6

Aplicacao web em React + TypeScript para criar, editar, salvar e rolar fichas de personagens de RPG de velho oeste inspirado no espirito do +2D6.

## Rodando

```bash
npm install
npm run dev
```

Para verificar a versao de producao:

```bash
npm run test
npm run build
```

O app salva fichas no `localStorage` do navegador. A exportacao em PDF e feita pelo proprio navegador: use o botao de impressao e escolha "Salvar como PDF".
